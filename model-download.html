Quick start

  1. Open the file and go to the Analysis tab.
  2. Edit only the shaded blue cells — your price, rent, rate, and expenses.
  3. Everything else (NOI, cap rate, cash-on-cash, DSCR, the 10-year projection, and return on sale) recalculates automatically.
  4. Using Google Sheets? Open the file, then File → Make a copy (or Save as Google Sheets) so your edits stick.
  5. See the How to use tab for what each metric means.
  6. The Income & Expense Tracker works the same way — enter your monthly rent and costs in the blue cells; the annual P&L and Summary tab total automatically. Sample data is included; clear it and use your own.
